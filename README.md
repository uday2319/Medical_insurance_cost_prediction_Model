# Medical_insurance_cost_prediction_app
 Here it calculates the medical insurance cost of a person in a single shot and
i have used RandomForestRegressor algorithm ,scikit-learn,pandas,numpy ,matplotlib ,seaborn and some more libraraies etc
and  used various evaluation metrics like MSE,accuracy, etc and  also model performance assesing metrics like cross validation and hyperparameter tuning(GridsearchCV) and also
 Deployed in streamlit
